st module¶

This is a minimal working version of sample strategy Current version: v1.0.0 Author: MY CAPITAL

st.main()¶

Main Entrance for local simulation, a simple local simulation loop will be run to verify logic of underlying strategy. It first initialize the simulation settings, then calls the simulation task.

>>> st1 = StratConfig.StratItem("sample_strat", stratid=888)

Construct a StratConfig.StratItem instance with name sample_strat and id 888.

>>> st1.contracts.append("sr|R1|CZCE|7|20|0|Account1")

“Name|Rank(R1/R2)|Exchange|FeedType|Maxpos(per order)|Account”

Subscribe contracts as specified by the formatted string. For multiple contracts, simply append more formatted string to the list.

>>> st1.accounts.append("Account1|999999.99|888888.88|CNY|1.0")

“Name|Available Cash|Asset Cash|Currency|Exchange Rate”

Similar to contracts subscription, simply append account to list.

>>> st1.output_file_path = "config/"
>>> st1.strat_param_file = "config/strat_param.txt"

output_file_path corresponds to a directory available for user to write into. strat_param_file is an external parameter file that a strategy might read from.

>>> cfg1 = StratConfig()

Initialize a StratConfig.StratItem instance

>>> cfg1.strat_item = st1.items()

Fill config with strategy.

>>> cfg1.start_date = 20170901
>>> cfg1.end_date = 20170910
>>> cfg1.begin_time = 85800000
>>> cfg1.end_time = 273000000
>>> cfg1.day_night_flag = 0

Simulation date time configuration.

>>> cfg1.time_interval = 1

Specifies the time interval between each on_timer.

>>> execute_task(cfg1.items())

Start task execution
